# Chat_Application
A console-based chat application built in C++ using Object-Oriented Programming and STL data structures. It supports user registration, secure login, sending messages, and reading inbox messages through a menu-driven interface. The project demonstrates concepts like encapsulation, queues, vectors, and maps in a practical system.

# C++ Chat Application

A console-based chat application developed using C++ that demonstrates Object-Oriented Programming (OOP) principles along with Standard Template Library (STL) data structures. The system allows users to register, log in, send messages, and read received messages through a menu-driven interface.

---

## Features

* User registration with username and password
* Secure login authentication
* Send messages between registered users
* Inbox system implemented using Queue (FIFO order)
* Display list of all users
* Modular OOP-based design

---

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL Containers (vector, queue, map)
* Console Input/Output

---

## Project Structure

* Message Class: Stores sender and message content
* User Class: Manages user credentials and inbox
* ChatSystem Class: Handles user management, authentication, and messaging
* Main Function: Provides the interactive menu-driven interface

---

## How to Run

### Compile the Code

```bash
g++ chat.cpp -o chat
```

### Run the Program

```bash
./chat
```

For Windows:

```bash
chat.exe
```
